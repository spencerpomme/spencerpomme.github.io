新年展望

昨天晚上临睡前被朋友圈里各种辞旧迎新和年终总结、新年展望所刷屏，心里不由得生出一丝感慨。往年自己也非常热衷此道，不过反而是成长最多的2016年让我没有什么兴致去「总结自己」，可能是因为成长让我更加真切地意识到了自己和真正的厉害的差距。不过更可能，只是因为觉得我所关注和在意的事情越来越难以为现在的朋友圈子所了解了

不过，「展望2017」这件事还是值得做一下的。隐约记得自己的2016目标大概只完成了80%左右，还有一两项明显有放卫星嫌疑的任务没能完成。其实归根结底还是高估了自己的努力程度和低估了其它事情对经历的牵扯。那么今年对自己的期望有：

  • 学习方向：
    1. 完成毕业论文并毕业
    2. 日语N2；
    3. 完成MIT OCW上线性代数剩余部分、概率统计和离散数学；
    4. 完成2017年书单；
    5. 完成操作系统、计算机网络这两门基础课；
    6. 上Andrew Ng的机器学习课并拿到证书；
  • 特定事件：
    1. 写一个个人作品集网站；
    2. 完成游戏剧本；
    3. 坚持每天写日记；
    4. 找到SDE实习；
  • 身体健康：
    1. 体重控制在75kg以下，且体脂率小于等于20%；
    2. 油炸食品等不健康垃圾食品食用次数控制在每月1～2次；
